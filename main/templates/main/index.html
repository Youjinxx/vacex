여기가 index.html 입니다.
templates/main 에서 main 안에 파일을 안넣어서 오류가 났었음.

여기에 post list를 나열하려고 함.
{% for p in posts %}

{{ p }}
{% endfor %}
//위와 같은 문구를 이용해서 반복문을 불러온다. 일반 변수는 % 없이 중괄호 두개를 이용하여 불러온다.

여기에 post list와 상세 정보를 나열하려고 함.
{% for p in posts %}
{{p.title}}
{{p.created_at}}
{{p.content}}
{% endfor %}

이 위치에 댓글을 넣으려 함
